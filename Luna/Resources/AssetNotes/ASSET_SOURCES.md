# Luna Asset Sources

Downloaded for Luna b14 on 2026-05-15.

## Planet And Moon Textures

The planet textures are compact JPEG maps from NASA/JPL Solar System Simulator texture maps. The Moon texture is the 2K color map from NASA Scientific Visualization Studio's CGI Moon Kit.

- `Textures/Planets/mercury_texture.jpg` - NASA/JPL Solar System Simulator, Mercury map, https://space.jpl.nasa.gov/tmaps/mercury.html
- `Textures/Planets/venus_texture.jpg` - NASA/JPL Solar System Simulator, Venus map, https://space.jpl.nasa.gov/tmaps/venus.html
- `Textures/Planets/earth_texture.jpg` - NASA/JPL Solar System Simulator, Earth map, https://space.jpl.nasa.gov/tmaps/earth.html
- `Textures/Planets/moon_texture.jpg` - NASA SVS CGI Moon Kit, 2K LROC color map, https://svs.gsfc.nasa.gov/4720/
- `Textures/Planets/mars_texture.jpg` - NASA/JPL Solar System Simulator, Mars map, https://space.jpl.nasa.gov/tmaps/mars.html
- `Textures/Planets/jupiter_texture.jpg` - NASA/JPL Solar System Simulator, Jupiter map, https://space.jpl.nasa.gov/tmaps/jupiter.html
- `Textures/Planets/saturn_texture.jpg` - NASA/JPL Solar System Simulator, Saturn map, https://space.jpl.nasa.gov/tmaps/saturn.html
- `Textures/Planets/uranus_texture.jpg` - NASA/JPL Solar System Simulator, Uranus map, https://space.jpl.nasa.gov/tmaps/uranus.html
- `Textures/Planets/neptune_texture.jpg` - NASA/JPL Solar System Simulator, Neptune map, https://space.jpl.nasa.gov/tmaps/neptune.html

## GLB Models

The satellite models are from NASA 3D Resources / NASA-3D-Resources on GitHub. NASA describes these assets as free to download and use, with use subject to NASA image and media usage guidelines.

- `Models/Moons/moon_small.glb` - NASA SVS Moon 3D Models for Web, AR, and Animation, https://svs.gsfc.nasa.gov/14959
- `Models/Satellites/terra.glb` - NASA 3D Resources, Terra, https://github.com/nasa/NASA-3D-Resources/tree/master/3D%20Models/Terra
- `Models/Satellites/aqua.glb` - NASA 3D Resources, Aqua (A), https://github.com/nasa/NASA-3D-Resources/tree/master/3D%20Models/Aqua%20(A)
- `Models/Satellites/tess.glb` - NASA 3D Resources, Transiting Exoplanet Survey Satellite (TESS) (A), https://github.com/nasa/NASA-3D-Resources/tree/master/3D%20Models/Transiting%20Exoplanet%20Survey%20Satellite%20(TESS)%20(A)
- `Models/Satellites/tdrs.glb` - NASA 3D Resources, Tracking and Data Relay Satellites (TDRS) (A), https://github.com/nasa/NASA-3D-Resources/tree/master/3D%20Models/Tracking%20and%20Data%20Relay%20Satellites%20(TDRS)%20(A)

## Usage Notes

NASA/JPL texture maps include representational and enhanced maps. They are appropriate for educational visualization, not scientific analysis.

Artificial satellite sizes in `celestial_bodies.json` are approximate display values until Luna gets a dedicated spacecraft dimensions model.
